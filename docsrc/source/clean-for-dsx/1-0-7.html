PixieDust release notes 1.0.7

Here's what's new in Release 1.0.7 (23 June 2017):

Bug fixes

  • Updated support for GraphFrames to account for new dependencies in recent GraphFrames versions ( #294 ).

Enhancements

  • The display() API can now automatically visualize properly formatted GraphFrame data as a star graph ( #354 ).

  • Region maps now take custom color ranges using the mapColorAxis parameter ( #371 ).

  • Bar charts now support custom ordering (ascending/descending), which users can optionally select via the display options menu ( #272 ).

  • Data table visualizations now format cells with URLs automatically as clickable links ( #376 ).
