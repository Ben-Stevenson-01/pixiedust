Here’s what’s new in Release 1.0.5 (19 May 2017):

Bug fixes

  • Chart size is now customizable with a slider bar for scaling images (#313).
  • Bar- and line-charts in matplotlib are now constrained with a maxiumum width. This fixes a bug where charts with over 200 data points would not render because they were too wide (#276).
  • PixieDust no longer shows redundant progress monitors when running in the IBM DSX environment, which already displays Spark job progress by defualt (#318).
